# 🚀 MotoCRM - Bike Sales Lead Management System

MotoCRM is a professional Lead Management System built for motorcycle dealerships. It helps sales teams manage leads, follow-ups, customer information, and sales activities through a clean and responsive dashboard.

---

## ✨ Features

- 📊 Interactive Dashboard
- 👥 Lead Management (Create, Update, Delete)
- 📅 Follow-up Management
- 📈 Reports & Analytics
- 🔍 Search & Filter Leads
- 📱 Responsive Design
- 🎨 Bootstrap 5 UI
- 🗂️ EJS Template Engine
- 🛢️ MongoDB Database
- ⚡ Express.js Backend

---

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/NamanJain19/MotoCRM.git
```

### Go to Project Folder

```bash
cd MotoCRM
```

### Install Dependencies

```bash
npm install
```

### Configure Environment Variables

Create a `.env` file:

```env
PORT=3000
MONGODB_URI=mongodb://127.0.0.1:27017/motocrm
```

### Start the Server

```bash
npm start
```

or

```bash
node server.js
```
